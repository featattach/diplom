Шаг 1. Скачайте шаблон или подготовьте свой файл

В шаблоне перечислены все заголовки: основные (название, модель, тип техники, серийный номер, расположение, статус, категория, описание, организация, пользователь) и технические характеристики (CPU, ОЗУ, тип/объём диска, IP адрес, мат. плата, ОС, блок питания, диагональ экрана/монитора, разрешение экрана, юниты (U), дата выпуска). Поддерживаются синонимы: «Имя», «Наименование», «Серийный», «Расположение», «Кабинет», «Организация», «Компания», «Пользователь», «Ответственный»; для техники — «Процессор»/«CPU», «Память»/«RAM»/«ОЗУ», «IP»/«Сетевая карта» → IP адрес, «Мат. плата»/«Материнская плата», «Дата производства» и др.

Поля с выбором (указывайте точно):
  • Статус — только: {{ status_options | join(", ") }}.
  • Тип техники — только: {{ equipment_kind_options | join(", ") }}. При опечатке значение не подставится (в карточке будет «—»).
  • Серийный номер — не повторяйте в файле и с уже имеющимися в базе, иначе строка будет пропущена.
  • Дата выпуска — формат ГГГГ-ММ-ДД или ДД.ММ.ГГГГ. Юниты (U) — целое число.
