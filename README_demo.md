# Описание проекта: система учёта оборудования

Веб-приложение для учёта оборудования, инвентаризации и отчётов. Реализовано на FastAPI, Jinja2 и Bootstrap. Учёт активов (ПК, ноутбуки, серверы, мониторы и т.д.), привязка к организациям, журнал событий (перемещения, списание), кампании инвентаризации с отметкой «найдено» по QR, отчёты и экспорт в Excel.

### Краткое описание функционала

**Учёт оборудования:**
- Ведение карточек активов: название, серийный номер, тип техники, расположение, статус (в работе / на ремонте / списано), организация, параметры (ОС, CPU, RAM и т.д.).
- Журнал событий по каждому активу: создание, изменение полей, перемещение, списание.
- Импорт списка оборудования из Excel, экспорт в Excel с учётом фильтров.

**Инвентаризация:**
- Кампании инвентаризации с привязкой к организации или без.
- Формирование объёма проверки (снимок списка активов на момент старта).
- Отметка «найдено» по карточке актива или по сканированию QR-кода.
- Завершение кампании, сводка (в объёме / найдено / не найдено), экспорт результатов в Excel.

**Отчёты:**
- Сводка по количеству активов и кампаний.
- Отчёт «Оборудование»: список с фильтрами (название, статус, тип, расположение, организация), экспорт в Excel с метаданными (кто сформировал, когда, фильтры).
- Отчёт «Светофор»: возраст техники по дате выпуска (зелёный / жёлтый / красный), экспорт в Excel.

**Организации (компании):**
- Справочник организаций, привязка активов к организации.
- Сводка по технике организации (количество по статусам и по расположению).

**Безопасность и администрирование:**
- Роли: **admin** (полный доступ, пользователи, бекапы), **user** (создание/редактирование активов, кампаний, экспорт), **viewer** (только просмотр).
- Сессия в cookie (CSRF-защита формы входа).
- Бекапы: создание zip (БД + avatars + qrcodes), скачивание, восстановление. Доступно только admin.

<!-- ВСТАВИТЬ СКРИНШОТ: детальная страница актива с полями и историей событий -->

![Карточка актива](image/asset_detail.png)
